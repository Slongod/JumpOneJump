# 蹦一蹦

> 王翼天 & 张锦岳
> 2025 年 12 月

本 DEMO 使用 `pygame` 库负责渲染和接收键盘输入，用自制的类实现了碰撞、动作等功能。

1. `gamemain.py` 是游戏主程序

   - `ad/左右 `移动，`空格` 跳跃

2. `mapmaker.py` 是地图制作工具：// 注意：该部分需要更新

   - `wasd` 移动当前选定的方块（按一下移动 1m）
   - `上下左右` 缩放当前选定的方块（按一下横向/纵向长度变化 1m）
   - `o `上一个方块，`p` 下一个方块
   - `i` 插入新的方块
   - 在程序退出的时候，自动将已经制作的地图写入 `map.move2dmap` 中。

3. `character.py` 是物理模拟/角色渲染类
   - `box` 类是用于碰撞检测的基础矩形属性
   - `player` 类继承自 `box`，添加了横向最大加速度、阻力加速度、纵向跳跃速度等，均可在调用初始函数的时候自定义。
   - `text` 类是文字类。
   - `xxx_painter` 类（其中 `xxx` 是某种类型）用于普通矩形或玩家形象转化为多边形。同时，`player_painter` 还额外支持落地/跳跃的形变和行走时的旋转。
   - `player_controller` 类用于接收玩家操作，同时实现了根据跳跃键的按键时长决定高度、跳跃缓冲和野狼时间。
